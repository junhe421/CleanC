# CleanC - 极简C盘清理优化工具

CleanC是一款专为电脑初学者（小白用户）设计的C盘清理与优化工具。它提供了比市面上同类产品更快速、界面更直观友好的体验，让用户能够安全、轻松地管理和释放C盘空间。

由 **Next Wave** 提供技术支持

![CleanC应用截图](assets/screenshot.png)

## 主要功能

### 智能扫描与安全分级
- 高速扫描引擎，快速分析C盘内容
- 清晰的图形化界面展示扫描结果
- 自动对文件进行安全等级划分（"可安全清理"、"建议保留"、"系统关键文件"）
- 通俗易懂的语言解释，防止误操作

### 一键快速清理
- 醒目且操作简单的"一键清理"按钮
- 默认只选中被标记为"可安全清理"的垃圾文件
- 实现秒级响应和快速空间释放

### C盘管理助手
- **大文件查找器**：列出占用空间最大的文件，方便用户识别并决定是否删除
- **重复文件查找器**：找出内容完全相同的文件，提供简单的清理选项
- **程序轻松卸载**：比系统自带更清晰、更彻底的程序卸载功能
- **开机加速**：用简单的开关方式，管理开机自启程序，加快开机速度

## 技术特点
- 使用Electron框架构建，确保高性能和跨平台兼容性
- 优化的文件扫描算法，提供最佳性能
- 简洁明了的用户界面，专为非技术用户设计
- 内置安全机制，防止误删重要文件

## 安装指南

### 系统要求
- Windows 10/11 (64位)
- 最小内存: 4GB RAM
- 磁盘空间: 100MB空闲空间

### 安装步骤
1. 从[官方网站](#)下载最新版CleanC安装包
2. 双击安装包运行安装程序
3. 按照屏幕提示完成安装
4. 安装完成后，桌面会创建CleanC快捷方式

## 使用说明

### 快速入门
1. 启动CleanC应用
2. 主界面显示C盘当前使用情况
3. 点击"开始扫描系统"按钮进行全盘扫描
4. 扫描完成后，勾选要清理的项目
5. 点击"清理所选项目"按钮开始清理

### 高级功能
- **查找大文件**：在左侧菜单选择"大文件查找"，设置最小文件大小后点击查找
- **查找重复文件**：在左侧菜单选择"重复文件"，选择要扫描的目录后点击查找
- **管理启动项**：在左侧菜单选择"开机加速"，使用开关控制各个应用的自启动状态
