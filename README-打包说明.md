# CleanC 打包与安装指南

本文档提供了如何将CleanC应用程序打包为安装程序并保护源代码的详细说明。

## 打包前准备

1. 确保安装了Node.js (推荐v16或更高版本)
2. 确保所有依赖都已安装：`npm install`
3. 确认assets文件夹中已包含以下文件：
   - `icon.svg` - 应用图标
   - `next-wave-logo.png` - 品牌logo
   - `qrcode-wechat.png` - 公众号二维码

## 打包方法

### 方法一：使用打包脚本（推荐）

1. 双击运行 `build-installer.bat` 批处理文件
2. 等待打包过程完成，这可能需要几分钟时间
3. 打包完成后，安装程序将位于 `dist` 目录中，文件名为 `CleanC-Setup-1.0.0.exe`

### 方法二：使用命令行

1. 打开命令提示符或PowerShell
2. 切换到项目根目录
3. 运行以下命令：
   ```
   npm run build:installer
   ```
4. 打包完成后，安装程序将位于 `dist` 目录中

## 安装程序功能

打包后的安装程序具有以下特点：

1. **源代码保护**：所有JavaScript代码已被打包为ASAR归档文件，增加了逆向工程的难度
2. **自定义安装选项**：用户可以选择安装位置
3. **快捷方式创建**：自动在桌面和开始菜单创建快捷方式
4. **许可协议**：包含最终用户许可协议
5. **品牌资源保留**：确保品牌logo和公众号二维码正确显示

## 安装后的防护措施

安装程序使用了以下技术来保护您的代码和资源：

1. **ASAR打包**：源代码被打包为ASAR格式，增加了逆向难度
2. **最大压缩**：使用了最高级别的代码压缩
3. **资源分离**：图像等资源文件与代码分开存储
4. **许可协议保护**：明确禁止反编译和逆向工程

## 注意事项

1. 不要分发源代码文件夹，只分发打包后的安装程序
2. 如果需要更新版本，请修改`package.json`中的版本号
3. 打包过程中会创建临时文件，但会在结束时自动清理
