# CleanC - 简单易用的C盘清理工具

CleanC是一款专为电脑初学者设计的C盘清理工具，通过简单直观的界面，帮助用户轻松释放C盘空间，保持系统流畅运行。

## 功能特点

- **磁盘空间分析**：直观显示C盘使用情况，了解空间分配
- **快速清理**：一键清理系统临时文件、浏览器缓存等垃圾文件
- **大文件查找**：快速定位占用大量空间的文件，方便管理
- **重复文件查找**：自动识别并列出内容相同的重复文件，减少空间浪费
- **程序卸载**：轻松卸载不需要的程序，释放更多空间

## 安装说明

### 方法一：使用安装脚本（推荐）

1. 解压下载的压缩包
2. 右键点击`install.bat`文件，选择"以管理员身份运行"
3. 按照提示完成安装
4. 安装完成后，通过桌面快捷方式启动程序

### 方法二：直接运行（无需安装）

如果您不想安装到系统目录，也可以直接运行程序：

1. 解压下载的压缩包到任意位置
2. 双击`start.bat`文件启动程序

## 系统要求

- Windows 10/11 操作系统
- 至少200MB可用磁盘空间
- 需要安装Node.js（如果没有安装，启动脚本会提示您安装）

## 首次使用

首次启动程序时，系统会自动安装必要的依赖，这可能需要几分钟时间，请耐心等待。安装完成后，后续启动将会更快。

## 常见问题

**Q: 启动程序时提示"需要安装Node.js"怎么办？**

A: 请访问[Node.js官网](https://nodejs.org)下载并安装最新的LTS版本。

**Q: 程序无法获取磁盘信息怎么办？**

A: 请确保以管理员身份运行程序，或尝试重新启动计算机后再试。

**Q: 清理后没有释放预期的空间怎么办？**

A: 某些系统文件可能无法直接清理，您可以尝试使用Windows自带的磁盘清理工具作为补充。

## 注意事项

- 使用"大文件删除"和"重复文件删除"功能时，请确认文件确实不需要再使用
- 程序默认不会删除系统关键文件，但删除前请仔细确认
- 建议定期使用本工具进行清理，保持系统流畅运行
